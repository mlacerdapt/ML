💡 Guia de Orientação Rápida

Método de Trabalho

Segue este guia sempre que clicares em "Novo Projeto" no teu CMS para manteres o foco, gerires o tempo de forma eficiente e garantires a máxima qualidade no teu portfólio.

1. Metadados e Briefing (O Arranque)

⏱️ 5 MINUTOS

O objetivo aqui é definir o limite do projeto para não te perderes.

  • Briefing: Máximo de 3 linhas. Responde apenas: O que é? Qual é o estilo? Qual o objetivo (ex: testar iluminação, peça para venda)?
  • Tags: Usa apenas 3 a 5 tags estratégicas (ex: #Japandi, #Vray, #BambuLab, #Interiores). Mais do que isso polui a pesquisa.
  • Status: Começa sempre em "Em andamento" e "Privado". Só mudas para "Público" quando estiveres 100% orgulhoso do resultado.

2. Referências e Moodboard (O Foco)

⏱️ Estritamente 20 MIN

A internet é um buraco negro de tempo. Esta é a fase onde deves ser mais implacável. Usa um cronómetro.

  • Quantidade de Imagens: Seleciona um máximo de 5 imagens.
  • 1 para iluminação/clima, 1 para paleta de cores, 2 para estética/arquitetura/detalhe, 1 planta ou croqui técnico (se aplicável).
  • Ação Final: Guarda-as na pasta do projeto gerada pelo CMS, atribui no painel e fecha o navegador. A partir desse minuto, a tua única referência visual são essas 5 imagens.

3. Campos Dinâmicos (A Execução)

⚙️ REGRAS

Regras para preencher a secção de dados consoante o que estás a produzir:

  • Para Renders: Define um limite à partida. Sugestão: 3 vistas (1 Ângulo Principal, 1 Close-up/Detalhe, 1 Vista Ortogonal ou Noturna). Não tentes renderizar a casa toda.
  • Para Impressão 3D: Tem o preço padrão dos teus rolos de filamento afixados perto de ti (ex: Bambu PLA Matte ~20€/kg) para inserires o custo de material em segundos com base na estimativa do Bambu Studio.
  • Para Vendas (Artigos): Define uma margem mínima inegociável (ex: no mínimo 50% de margem sobre o custo total, incluindo desgaste da máquina e eletricidade). Se o cálculo mostrar menos, o projeto precisa de reengenharia geométrica.

4. Gestão de Horas (O Cronómetro)

⏳ META: 2H/DIA

Esta é a tua bússola de produtividade.

  • A Regra: Regista o tempo no final de cada sessão diária. Se trabalhaste 1h45m, anota 1.75.
  • A Meta das 2h: O indicador visual não serve para te punir, mas para te obrigar a tomar decisões técnicas mais rápidas. Se o V-Ray está a demorar demasiado a afinar um reflexo, simplifica o material. A perfeição é inimiga do portfólio feito.

5. Galeria de Imagens (A Montra)

🖼️ MAX: 6-8 IMGS

Não sobrecarregues a página do cliente. A tua arte 3D precisa de espaço para "respirar" no ecrã.

  • Capa (1 Imagem): A "Mona Lisa". A imagem com melhor iluminação e composição.
  • Processo (1 a 2 Imagens): O breakdown. Pode ser metade clay / metade fotorrealista, ou uma foto da peça a sair da impressora.
  • Galeria Final (3 a 4 Imagens): Vistas complementares.
  • Minimalismo: Mesmo com o script Pillow a comprimir tudo perfeitamente, o minimalismo do teu design exige pouca quantidade e altíssima qualidade.
